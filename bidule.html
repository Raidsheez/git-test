  • "ls" pour afficher les dossier et les fichier (et les cachés)
  • Rentre dans le bon dossier avec "cd" et ressortir avec "cd ../"
  • "add + nomduficher" pour ajouter le fichier
  • "commit -m"message" pour mettre un message à la sauvegarde
  • "status" pour vérifier si tout est bien sauvegarder
  • "log" liste de tout les changement des arbres de travail
  • "checkout + numérodelasauvegarde" pour switcher entre les différents points de sauvegarde
